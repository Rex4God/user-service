# User Service

Includes API Server utilities:

## Dependencies

- MySQl
- Sequelize

## Backend

    
    - Language : Nodejs (Xpress framework)

## Database

    kind: MySQL
    db-name : user-service

## Setup

```
Clone repository from Github
```
Create .env  copy from .env.example
```
 npm install : To install the dependencies
```

## Lint

```
npm run lint
```


## Test

```
npm run test
```
